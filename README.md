# Magus

## Setup

Create a virtual environment and install dependencies:

- Linux
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
pip install -r requirements-dev.txt 
```

- Windows
```bash
python -m venv .venv
.venv/Scripts/activate
pip install -r requirements.txt
pip install -r requirements-dev.txt 
```
